# AI_Consultant
Responsible AI Consultant for Churn Strategy

Churn prediction
1. Reasoning layer
2. Strategy tradeoff analysis
3. What-if simulator
4. AI ethics + bias audit

Core user flow
- User uploads a customer dataset.
- App predicts churn risk.
- App explains key churn drivers.
- App audits fairness across sensitive or business-relevant groups.
- App recommends strategies.
- User tests “what-if” scenarios.
- App generates a consultant-style report.
